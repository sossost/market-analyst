# 지정학 이벤트 스냅샷 — 2026년 3월 21일

- **2026년 2월 28일**: 미국-이스라엘이 이란에 대한 공습 개시 (Operation Epic Fury). 군사시설, 핵시설, 지도부 타격.
- **2026년 3월 4일~**: 이란이 호르무즈 해협 "봉쇄" 선언. 3월 12일까지 상선 21척 공격 확인.

- **US 대중국 수입**: 2025년 6월 기준 전년 대비 약 절반 수준으로 감소 (PIIE 추산)
