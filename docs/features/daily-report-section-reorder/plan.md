# Plan: 데일리 리포트 MD 섹션 순서 재배치

## 문제 정의

현재 데일리 리포트 MD 파일 섹션 순서가 매크로 → 종목 → 매크로로 흐름이 끊긴다.
시장 온도/섹터 분석 후 바로 개별 종목 상세가 나오면서 거시적 흐름 파악이 중단됨.

## Before → After

### Before (현재)
1. 시장 온도 근거 (매크로)
2. 섹터 RS 랭킹 (매크로)
3. **특이종목 상세** (개별 종목 ← 흐름 끊김)
4. **주도주 예비군** (개별 종목)
5. 시장 흐름 및 종합 전망 (매크로 종합)
6. 전일 대비 변화 요약 (비교)
7. 관심종목 현황 (포트폴리오)

### After (제안)
1. 시장 온도 근거 (매크로 — 오늘 시장이 어떤가)
2. 섹터 RS 랭킹 (매크로 — 어디가 강한가)
3. 전일 대비 변화 요약 (비교 — 어제와 뭐가 달라졌나)
4. 시장 흐름 및 종합 전망 (종합 — 구조적으로 어떤 상황인가)
5. 관심종목 현황 (포트폴리오 — 추적 종목 상태)
6. ---  📋 부록: 종목 상세 (구분선)
7. 특이종목 상세 (참고 — 드릴다운용)
8. 주도주 예비군 (참고 — 드릴다운용)
