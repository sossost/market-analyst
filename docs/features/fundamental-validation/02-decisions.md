# Decisions: 펀더멘탈 검증 + 종목 리포트 시스템

## Decision 1: 검증 위치 — 토론 멤버 vs 별도 단계

**선택: 별도 검증 단계 (토론 후)**

| 옵션 | 장점 | 단점 |
|------|------|------|
| A. 토론 5번째 멤버 | 토론에 펀더멘탈 관점 추가 | 종목 개별 분석 ≠ 매크로 토론, 성격 불일치 |
| **B. 토론 후 검증** | 관심사 분리, 종목 단위 심층 분석 가능 | 토론에 펀더멘탈 관점 부재 |

근거: 토론은 시장 방향성/섹터 로테이션을 다루고, 펀더멘탈은 개별 종목 수준의 검증이므로 분리하는 것이 자연스러움.

## Decision 2: 데이터 소스 전략

**선택: 기존 DB 우선 + FMP 보충 (Phase 2만)**

- `quarterly_financials`: EPS, 매출, 순이익, FCF — 이미 수집 중
- `quarterly_ratios`: 마진, 밸류에이션 — 이미 수집 중
- FMP: 어닝 서프라이즈, 기관 보유 — Phase 2 종목만 주 1회

전 종목 FMP 수집은 불필요. Phase 2 = ~200종목 → API 비용 최소.

## Decision 3: 스코어링 방식

**선택: 정량 로직 (LLM 불필요)**

Minervini SEPA 기준은 명확한 숫자 기반이므로 LLM 판단이 아닌 정량 함수로 구현.
LLM은 점수 해석 + 내러티브 생성에만 사용.

## Decision 4: 아웃풋 체계

**선택: 주간 리포트 보조 + 종목 리포트 (A급)**

| 등급 | 주간 리포트 | 종목 리포트 |
|------|-----------|-----------|
| A | 펀더멘탈 보조 정보 포함 | 발행 |
| B | 펀더멘탈 보조 정보 포함 | 미발행 |
| C | "기술적으로만 Phase 2" 경고 | 미발행 |
| F | "펀더멘탈 미달" 경고 | 미발행 |

## Decision 5: 펀더멘탈 애널리스트 역할

**선택: 해석 전문가 (스코어링 결과를 내러티브화)**

정량 점수를 받아서:
- "EPS 3분기 연속 가속, 이익률 확대 동반 — 구조적 성장 시그널"
- "매출은 성장하나 이익률 역행 — 가격 경쟁 압박 우려"
같은 투자 의미를 부여하는 역할.
