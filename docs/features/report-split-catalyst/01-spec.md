# Feature Spec: 일간/주간 리포트 분리 + 카탈리스트 분석

**Status:** Confirmed
**Created:** 2026-03-04
**Author:** brainstorm session

---

## Overview

매일 종목을 추천하는 단일 에이전트를 **일간 시장 브리핑**과 **주간 종목 발굴**로 분리한다.
Brave Search API를 연동하여 특이종목의 카탈리스트(뉴스/이벤트)를 분석하고,
리포트를 Markdown 파일로 생성하여 Discord에 첨부 업로드한다.

## User Goals

- 투자자로서, 매일 시장 온도와 특이 이벤트만 빠르게 확인하고 싶다
- 투자자로서, 주말에 한 주간의 Phase 2 주도주를 심층 분석 받고 싶다
- 투자자로서, 급등/급락 종목의 원인(카탈리스트)을 즉시 파악하고 싶다
- 투자자로서, 표와 포맷이 깨지지 않는 깔끔한 리포트를 받고 싶다

## Behavior

### 일간 리포트 (월~금, 장 마감 후)

#### Happy Path

1. ETL 완료 후 에이전트 실행
2. 시장 브레드스 데이터 조회 (Phase 2 비율, 전일 대비 변화)
3. 주도 섹터/업종 변동 확인
4. 특이종목 스크리닝 (복합 조건)
   - 일일 등락률 ±5% 이상
   - 거래량 평균 대비 2배 이상
   - Phase 전환 (1→2 또는 2→3)
   - 위 조건 중 2개 이상 충족
5. 특이종목 발견 시 → Brave Search로 카탈리스트 검색 (종목당 1회)
6. 뉴스 1~3건 요약하여 카탈리스트로 기재
7. 리포트 작성 → MD 파일 생성 → Discord 전송 (요약 메시지 + 파일 첨부)
8. 리포트 이력 저장

#### 특이사항 없는 날

시장 온도 1~2줄 요약만 전송. MD 파일 첨부 없음.

```
📊 시장 일일 브리핑 (2026-03-04)

🌡️ 시장 온도: 보합
- Phase 2 비율: 32.1% (전일 대비 -0.2%)
- 특이사항 없음
```

#### 특이사항 있는 날 — 리포트 포맷

Discord 메시지 (요약):
```
📊 시장 일일 브리핑 (2026-03-04)

🌡️ 시장 온도: 강세
- Phase 2 비율: 34.5% (전일 대비 +1.2%)
- 주도 섹터: Technology (RS 85.2), Healthcare (RS 78.1)

🔥 특이종목 2건 — 상세 분석은 첨부 파일 참조
```

MD 파일 첨부 (상세):
```markdown
# 시장 일일 브리핑 (2026-03-04)

## 시장 개요

| 지표 | 값 | 전일 대비 |
|------|-----|-----------|
| Phase 2 비율 | 34.5% | +1.2% |
| 주도 섹터 | Technology | RS 85.2 |

## 특이종목

### 1. NVDA (+8.2%) | Technology | RS 82 | Phase 1→2 전환

- 거래량: 평균 대비 2.5배
- 카탈리스트: AI 신규 칩 발표 (Reuters), GTC 컨퍼런스 호재 (Bloomberg)

### 2. CRWD (+6.1%) | Cybersecurity | RS 75 | Phase 2

- 거래량: 평균 대비 3.1배
- 카탈리스트: FedRAMP 인증 획득 (PR Newswire)
```

### 주간 리포트 (토요일 오전)

#### Happy Path

1. 금요일 장 마감 데이터 기반으로 실행
2. 시장 브레드스 주간 변화 분석
3. Phase 2 종목 스크리닝 (RS 60 이상)
4. Phase 1→2 신규 전환 종목 우선 선별
5. 후보군 각각 Brave Search로 카탈리스트 검색
6. 핵심 주도주 3~8개 선정 + 심층 분석
7. 리포트 작성 → MD 파일 생성 → Discord 전송 (섹션별 분할 메시지 + 파일 첨부)
8. 리포트 이력 저장

#### 주간 리포트 포맷

Discord 메시지 (분할 전송):
- 메시지 1: 시장 주간 개요
- 메시지 2: 주도주 리스트 요약
- 메시지 3: Agent 인사이트
- 파일 첨부: 전체 상세 분석 MD

MD 파일 첨부 (상세):
```markdown
# 주간 시장 분석 리포트 (2026-02-24 ~ 2026-02-28)

## 시장 개요

| 지표 | 이번 주 | 지난 주 | 변화 |
|------|---------|---------|------|
| Phase 2 비율 | 34.5% | 32.1% | +2.4% |
| Phase 1→2 전환 | 12종목 | 8종목 | +4 |

## 주도 섹터

| 섹터 | RS | Phase 2 비율 | 주간 변화 |
|------|-----|-------------|-----------|
| Technology | 85.2 | 42% | +3.1% |
| Healthcare | 78.1 | 38% | +1.5% |

## 주도주 심층 분석

### 1. NVDA | Technology | RS 82 | Phase 2 초입

| 항목 | 값 |
|------|-----|
| 주간 등락률 | +12.3% |
| 거래량 (주간 평균 대비) | 2.1배 |
| MA150 기울기 | +0.8% |
| 52주 고점 대비 | -5.2% |

**카탈리스트:**
- AI 신규 칩 발표 (Reuters, 2/25)
- GTC 컨퍼런스 호재 (Bloomberg, 2/27)

**분석:** Phase 1→2 전환 직후 RS 급등. 섹터 전반 강세와 맞물려 추세 지속 가능성 높음.

### 2. ...

## Agent 인사이트

[시장 관찰, 섹터 로테이션 동향, 주의사항]
```

### Error Cases

- **Brave Search API 실패**: 카탈리스트 없이 기술적 데이터만으로 리포트 작성. "카탈리스트 조회 실패" 표기.
- **Discord 파일 업로드 실패**: 텍스트 메시지로 폴백. 로컬 파일 저장.
- **특이종목이 너무 많은 날 (10개+)**: 상위 5개만 카탈리스트 분석. 나머지는 리스트만.

### Edge Cases

| Situation | Expected Behavior |
|-----------|-------------------|
| Phase 2 종목 0개 (주간) | "신규 Phase 2 전환 종목 없음" + 시장 개요만 |
| 비거래일 (공휴일) | "거래일이 아닙니다" 한 줄 전송 |
| Brave Search 무료 한도 소진 | 카탈리스트 없이 기술적 데이터만으로 리포트 |
| MD 파일 8MB 초과 (Discord 제한) | 발생 가능성 극히 낮으나, 초과 시 텍스트 분할 전송 |
| 토요일에 금요일 데이터 미반영 | ETL 완료 확인 후 실행. 미완료 시 재시도 or 스킵 |

## Interface Design

### 새로운 도구 (Agent Tools)

#### search_catalyst

종목 티커를 받아 Brave Search API로 최근 뉴스를 검색하고 요약 반환.

```typescript
Input: { ticker: string, companyName: string }
Output: { results: Array<{ title: string, source: string, summary: string }> }
```

#### send_discord_report (기존 수정)

텍스트 메시지 전송 + MD 파일 첨부 지원.

```typescript
Input: {
  message: string,          // Discord 텍스트 메시지
  markdownContent?: string, // MD 파일 내용 (있으면 파일 첨부)
  filename?: string         // 파일명 (예: "daily-2026-03-04.md")
}
```

### 환경변수 추가

```
BRAVE_API_KEY=BSA...   # Brave Search API key
```

### 스케줄

| 리포트 | cron (UTC) | 설명 |
|--------|------------|------|
| 일간 | `0 0 * * 1-5` | 월~금 UTC 00:00 (KST 09:00) |
| 주간 | `0 1 * * 6` | 토 UTC 01:00 (KST 10:00) |

## Acceptance Criteria

- [ ] 일간 리포트: 시장 온도 + 특이종목 카탈리스트 분석 전송
- [ ] 주간 리포트: Phase 2 종목 발굴 + 카탈리스트 + 인사이트 전송
- [ ] 특이종목 없는 날 간단 요약만 전송
- [ ] Brave Search API로 카탈리스트 검색 동작
- [ ] Brave Search 실패 시 기술적 데이터만으로 폴백
- [ ] MD 파일 첨부하여 Discord 전송
- [ ] 표/포맷이 MD 파일에서 정상 렌더링
- [ ] 주간 리포트 섹션별 분할 메시지 전송
- [ ] GitHub Actions 일간/주간 스케줄 분리
- [ ] 기존 일간 리포트 이력 로직 유지

## Scope

**In Scope:**
- 시스템 프롬프트 일간/주간 분리
- Brave Search API 연동 (search_catalyst 도구)
- Discord Webhook multipart/form-data MD 파일 첨부
- 메시지 분할 전송 로직
- GitHub Actions 주간 스케줄 추가
- 특이종목 복합 조건 스크리닝 (등락률 + 거래량 + Phase)

**Out of Scope:**
- 실적 캘린더 연동
- 섹터별 심층 트렌드 분석
- 모바일/웹 알림 푸시
- 리포트 히스토리 웹 대시보드
- Discord Embed 디자인
